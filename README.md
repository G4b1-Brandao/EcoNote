# ♻️ EcoNote – Reaproveitamento de notebooks Descartados para o Ambiente Acadêmico  

Este repositório contém o desenvolvimento do projeto **EcoNote**, criado para a disciplina **Análise e Projeto de Sistemas**. O objetivo do projeto é promover a **inclusão digital** através do **reaproveitamento de notebooks descartados**, conectando doadores a estudantes que necessitam desses equipamentos para seus estudos.

## 🎯 Público-Alvo  
Nosso público-alvo são **estudantes do ensino superior e técnico do IFCE Campus Aracati** que enfrentam dificuldades financeiras para adquirir um notebook. Além disso, o projeto busca engajar **universidades, instituições de ensino, professores e alunos** que desejam contribuir com a inclusão digital por meio da **doação de equipamentos**.  

## 🚀 Como Funciona?  
- ✅ **Cadastro de Usuários** (Alunos e Professores) com validação por tipo de e-mail institucional  
- ✅ **Login com autenticação por grupo** (Aluno ou Professor)  
- ✅ **Cadastro e Gerenciamento de Notebooks** (feita por administradores)  
- ✅ **Solicitação de Notebooks** feita por alunos  
- ✅ **Avaliação das Solicitações** feita por professores  
- ✅ **Indicação de Alunos** para prioridade de recebimento  
- ✅ **Controle de Devoluções e Manutenções**  

> *Obs.: Administradores têm acesso privilegiado para gestão do sistema, mas não realizam cadastro direto na plataforma.*  


## 🔧 Tecnologias Utilizadas 
- **Linguagem:** Python

- **Banco de Dados:** SQLite3 (desenvolvimento)
